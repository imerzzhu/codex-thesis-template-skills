# Open Source Policy

This repository is a safe public package of reusable Codex skills for thesis and defense-material formatting.

## Included

- Standard `SKILL.md` files with public skill names.
- `agents/openai.yaml` metadata for each skill.
- Condensed formatting rules derived from local template work.
- Python audit and repair scripts that do not depend on private filesystem paths.
- MIT license and project documentation.

## Excluded

- School template files in `.doc`, `.docx`, `.dotx`, `.ppt`, or `.pptx` formats.
- User thesis, dissertation, defense, figure, table, or appendix files.
- Rendered pages, generated audit reports, and local work outputs.
- Private filesystem paths, credentials, secrets, and local cache paths.
- Generated bytecode, caches, backup folders, and temporary files.

## Contribution Rules

- Keep each skill self-contained and compatible with the Codex Skills structure.
- Keep `SKILL.md` frontmatter limited to standard fields.
- Put detailed reusable rules in `references/` and deterministic helpers in `scripts/`.
- Do not add template assets unless their redistribution rights are explicit and documented.
- Do not add private paths, credentials, generated user documents, or cache artifacts.
- Run the structure, safety, and script validation checks before publishing changes.
