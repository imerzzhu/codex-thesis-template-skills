## Local Use

1. Install or clone this repository.
2. Obtain the template files from an authorized school or project source.
3. Place each file at the matching path above.
4. Keep large work outputs outside the repository, for example under `D:\Cache\Dev\thesis-template-skills`.

The MIT license in this repository covers only repository code and documentation. It does not cover template files that users add locally.
