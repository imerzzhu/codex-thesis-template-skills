# CDUT English Thesis Format Rules

These rules are extracted from the supplied Chengdu University of Technology English-major bachelor thesis templates. When the template file and this summary disagree, inspect the template asset and follow the template.

## Document Setup

- Paper: A4 portrait.
- Main template sections use A4 page size. Margins in the visible instructions are generally top/bottom/left/right about 2.54 cm, header about 1.5 cm, footer about 1.75 cm. Converted template sections may show 2.4 cm margins in front matter and wider left/right margins in later sections.
- Body line spacing: 1.5 lines.
- Character spacing: default scale 100%, standard spacing.
- Page numbers: small 5-point style, Times New Roman, centered at the bottom.
- Running header begins from the English abstract page: `成都理工大学20xx届学士学位论文`, Songti/SimSun small 5, centered.

## Page Numbering And Order

- English abstract page: roman page number `i`.
- Chinese abstract page: roman page number `ii`.
- Contents page: no visible page number and does not list `Contents` as a TOC entry.
- Arabic page numbering starts from `Introduction` as page 1.
- `Acknowledgements`, optional `Notes`, and `Works Cited` start on new pages.
- Required order: cover, English abstract, Chinese abstract, Contents, thesis body from Introduction, Conclusion, Acknowledgements, optional Notes, Works Cited.

## Cover

- Remove highlighted or blue instructional placeholder text before final printing.
- English title: Times New Roman, size 2 Chinese size equivalent, bold, centered.
- Chinese title: Heiti, size 2 Chinese size equivalent, centered.
- Required metadata includes `By`, student name, `Supervised by`, supervisor title/name, date, `College of Foreign Languages and Cultures`, and `Chengdu University of Technology`.
- Supervisor title choices in the template include `Prof.`, `Assoc. Prof.`, `Lecturer`, `Mr.`, and `Ms.`.
- Date examples use English month/day/year format such as `May 22nd, 2024`; keep ordinal suffix formatting when used.

## English Abstract

- English thesis title: Times New Roman, size 2, bold, centered.
- Student/Class/Supervisor metadata: Times New Roman, small 4, bold, centered. `Class` is the class number, not the student ID.
- `Abstract`: centered, second-level heading, small 2, bold.
- Leave a blank line between the `Abstract` heading and content.
- Abstract body: Times New Roman, small 4, about 250-300 words.
- `Key words:` uses two words, small 4, bold. Use 3-5 keywords; ordinary words are lowercase and separated with semicolons.

## Chinese Abstract

- Chinese title: Heiti, size 2, centered.
- Author/class/supervisor line: Songti, small 4, centered; use the complete class number.
- Heading: `摘  要`, centered, small 2, Heiti, with visible spacing between `摘` and `要`.
- Chinese abstract must correspond closely to the English abstract and avoid omissions or mismatched meaning.
- `关键词`: label in small 4 Heiti, no first-line indent; keyword text in small 4 Songti. Use 3-5 keywords corresponding to the English keywords and separate them with Chinese semicolons.

## Contents

- `Contents`: centered, second-level heading, small 2, bold.
- TOC entries use small 4; second-level headings are bold.
- The Contents page should be generated from headings where possible and should fit one page if feasible.
- Do not include `Contents` itself as a TOC entry.
- Typical entries include Abstract, 摘要, Introduction, numbered chapters, Conclusion, Acknowledgements, Notes, and Works Cited.

## Body And Headings

- Main body text: Times New Roman, small 4. Chinese body text or labels: Songti, small 4.
- Introduction: unnumbered, no punctuation after the heading, left aligned, small 2, bold, 0.5 line spacing before.
- Numbered chapter headings: e.g. `1. Trademark and Trademark Translation`, left aligned, small 2, bold, 0.5 line spacing before.
- Third-level headings: e.g. `1.1 XXXX of Trademark`, small 3, bold, 0.5 line before and after.
- Fourth-level headings: e.g. `1.1.1 Easiness in Reading and Memorizing`, small 4, bold, 0.5 line before and after.
- Conclusion: unnumbered, no punctuation after the heading, left aligned, small 2, bold, 0.5 line spacing before.
- The main sample expects four chapters with about three subsections each, but preserve the user's actual approved thesis structure unless asked to restructure.

## Tables And Figures

- Tables are numbered continuously by chapter, e.g. `表2-5`; do not repeat or skip numbers.
- Table number/title: centered above the table, size 5, bold.
- Figures are numbered continuously by chapter, e.g. `图2-5`; do not repeat or skip numbers.
- Figure number/title: centered below the figure, size 5, bold.
- If there is only one figure, the template allows `附图`.
- Keep table and figure terminology, symbols, and units consistent with the text.
- Tables should have clear structure, columns with quantities/units, vertically aligned numbers, and no `同上` or `同左`.
- Leave appropriate blank space before and after tables and figures.

## Notes

- `Notes` is optional. If used, start it on a new page.
- The `Notes` heading is unnumbered, no punctuation after the heading, left aligned, small 2, bold, 0.5 line spacing before.
- Note markers in the thesis body use Arabic superscript numbers.
- Each note first line indents four characters; following lines align flush.

## Works Cited

- `Works Cited` starts on a new page.
- Use the exact plural title `Works Cited`.
- Heading is unnumbered, no punctuation after the heading, left aligned, small 2, bold, 0.5 line spacing before.
- Reference entries use size 5 font.
- English references precede Chinese references.
- Sort references alphabetically by author surname within their language group.
