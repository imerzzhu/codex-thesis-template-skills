# SWPU Thesis Format Rules

These rules are extracted from `assets/swpu-eei-undergraduate-thesis-template.docx`. When this summary and the template differ, inspect the template asset and follow the template.

## Page Setup

- Template length: 20 rendered pages.
- Paper: A4 portrait.
- Sections: 4 section-property blocks in the supplied template.
- Page size: approximately `11906 x 16838` twips.
- Margins: top 2.5 cm, bottom 2.5 cm, left 3.0 cm, right 2.5 cm.
- Header distance: 1.5 cm.
- Footer distance: 1.5 cm.

## Core Typography

- Normal style: 10.5 pt baseline.
- Body style: small 4 Songti/SimSun, first-line indent 2 Chinese characters, exact 22 pt line spacing.
- Chinese abstract heading `摘  要`: size 3 Songti, bold, centered.
- English abstract heading `Abstract`: size 3 Times New Roman, centered.
- Chinese and English abstract body text: small 4; Chinese uses Songti, English uses Times New Roman.
- `关键词` / `Keywords`: small 4, bold label; keyword content small 4; terms separated by semicolons.

## Heading Styles

- Level 1 heading: size 3 Songti, flush left, bold.
- Centered level 1 heading: size 3 Songti, centered, bold. Used for `谢 辞`, `参考文献`, and `附 录`.
- Level 2 heading: small 3 Songti, flush left, bold.
- Level 3 heading: size 4 Songti, flush left, bold.
- Level 4 heading: small 4 Songti, flush left, bold.
- Level 5 heading: first-line indent 2 characters, small 4 Songti.
- Avoid orphan headings at the bottom of a page.
- If a heading has only one child heading, do not create that child level. For example, do not use only `1.1` without `1.2`.

## Contents

- `目  录`: size 3 Songti, bold, centered.
- Contents entries: small 4 Songti.
- TOC 1/2/3 styles use exact 22 pt line spacing.
- TOC level 2 is indented; TOC level 3 is further indented.

## Tables

- Every table must have its own table number and title.
- Table caption appears above the table and centered.
- Caption format: `表2.1 表题`, with the table number and title separated by one space.
- Table numbers are chapter-based and continuous, e.g. `表2.1`, `表3.1`.
- If a table continues on the next page, repeat the table header and place `续表××` at the upper right.
- Table paragraphs use single line spacing, centered alignment, and no first-line indent.
- Template notes allow table row spacing with minimum 0.8 cm when needed.

## Figures

- Figures must be carefully prepared, clean, and visually balanced.
- Figure caption appears below the figure and centered.
- Figure numbers are chapter-based and continuous, e.g. `图1.1`, `图1.2`.
-正文 should refer to every figure, table, and formula, e.g. `如图2.1所示`, `如表2.1所示`, `如公式（2.1）所示`.

## Formulas

- Formulas are centered.
- Formula numbers use parentheses and are placed at the right end of the line.
- Do not add dotted or dashed leaders between the formula and its number.
- Formula numbers are chapter-based and continuous, e.g. `（1.1）`, `（1.2）`, `（2.1）`.

## References

- `参考文献` uses the centered level 1 heading style.
- Undergraduate theses should have more than 20 references, including foreign-language references.
- Every listed reference must be cited in the thesis body.
- Book format: `[序号]主要责任者.文献题名.出版地：出版者，出版年.起止页码`
- Journal format: `[序号]主要责任者.文献题名.刊名，年，卷（期）：起止页码`
- Other sources follow the supplied examples, including standards such as `[4]GB151-89.钢制管壳式换热器`.
